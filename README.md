## Description

This application features a blog site where users can create posts and others can comment on these posts. Users are able to log into their account and see their posts from their dashboard. 

## Installation

- MYSQL
- Node.js
- NPMs:
    - Bycrpt
    - Express
    - Express-Handlebars
    - Express-Session
    - Dotenv
    - Mysql2
    - Sequelize
    - Nodemon
